#  Rocket Pizza

## Descripción
Rocket Pizza es una aplicación web desarrollada con Next.js que simula la página de una pizzería.  
Permite navegar entre distintas secciones como el menú, contacto y sucursales, además de visualizar el detalle de cada pizza mediante rutas dinámicas.

## Funcionalidades
- Página de inicio donde vemos el Nav y el Footer
- Sección de carta con categorías
- Listado de pizzas ya que hambuergeusas no esta disponible
- Detalle de cada pizza 
- Página de contacto con redes sociales

## Tecnologías utilizadas
- Next.js
- React
- Tailwind CSS
- JavaScript
